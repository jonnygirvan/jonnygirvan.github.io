Aviation Weather

Application Background

This is a personal project I am working on. The primary audience for this application are new pilots in training. My goal is to provide an easy to digest decoding of METAR reports for pilots who are starting the journey of obtaining a PPL, that gets them comfortable with reading reports in the long term. The idea is that a user will be able to pull in the latest report for a given airfield, and see a breakdown of the current conditions in a way that makes sense to them. A student should not rely on this forever though, so I want to provide multiple views of the report, so that as a student progresses they can switch the view up as they become familiar with the format of a METAR readout.

There are many applications that offer METAR services, but I wanted to create something that was less intimidating for a new student, as well as something that works cross platform.

This project is a topic that interests me as a student pilot, and is allowing me to bring together all the skills I have learned in the past into one solution.

Solution Process

Currently the application is in the beginning phases of development. The goal is to create an MVP that will be used to gather feedback for revisions and to drive the feature roadmap.

  1. MVP wireframes created in Figma
  2. Application styleguide created
  3. Creation of supporting 3D assets and animations using Blender
  4. MVP developed using a React frontend connected to the Aviavtion Weather Center to pull in data
  5. Build analytics into MVP and create a usability study to evaluate success of the current state
  6. Create feature and release roadmap for stable releases
